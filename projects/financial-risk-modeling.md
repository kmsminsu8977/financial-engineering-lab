## Category Scope

이 카테고리는 금융시장 리스크를 정량적으로 해석하고 모델링하는 연구를 다룹니다. 이론적 모델 이해와 실증 분석, 계산 구현을 함께 포함하는 형태를 목표로 합니다.

주요 주제 예시는 다음과 같습니다.

- 금리 예측 모델
- 자산가격 모형 분석
- 금융공학 기반 수리 모델 구현
- VaR 계산기
- 시뮬레이션 기반 리스크 연구

## Intended Portfolio Output

향후 이 카테고리의 각 프로젝트는 다음 요소를 기준으로 정리할 예정입니다.

- 문제 정의
- 데이터 또는 입력 변수 설명
- 사용한 모델과 방법론
- 결과 해석
- 한계와 확장 방향
